# PulseOS

A small, basic OS which does nothing..


# How to build the Kernel

Requirements:  Python 3.13.3, i686-elf-gcc, i686-elf-as, (QEMU or any other Virtualization Software)

Run "build.py all"
Then run "build.py run"
